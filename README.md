# DNS Lookup & IP Information Tool 🌐

This project is a Python-based networking tool that converts domain names into IP addresses and retrieves network-related information such as ISP, city, country, and organization.

## Features
- DNS lookup
- Reverse DNS lookup
- IP information (ISP, location)
- Streamlit user interface

## Tech Stack
Python, Streamlit, Socket, Requests API

## Run Locally

pip install -r requirements.txt
streamlit run app.py
